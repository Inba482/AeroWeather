# 🌦️ AeroWeather - Real-Time Weather Dashboard

A modern, responsive Weather Dashboard built using **HTML5, CSS3, and JavaScript (ES6+)** that provides real-time weather information for cities worldwide using the **OpenWeatherMap REST API**.

## 🚀 Project Overview

AeroWeather demonstrates the practical implementation of:

* Asynchronous JavaScript using **Fetch API** and **Async/Await**
* REST API integration
* Dynamic JSON data processing
* Real-time weather retrieval by city name
* Comprehensive error handling
* Responsive and modern user interface design

The application allows users to search for any city and instantly view current weather conditions, temperature, humidity, wind speed, pressure, and visibility.

## 🌐 API Integration

This project uses the OpenWeatherMap Current Weather API.

API Endpoint:

https://api.openweathermap.org/data/2.5/weather

Weather data is retrieved dynamically using asynchronous JavaScript and rendered in real time.

## ⚙️ Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/weather-dashboard.git
```

### 2. Open the Project

Open `index.html` in your browser.

### 3. Configure API Key

1. Create a free account on OpenWeatherMap.
2. Generate an API key.
3. Open the Settings panel inside the application.
4. Enter your API key and save.

### 4. Start Searching

Search for any city and view live weather information instantly.
